Tooltip 文字提示 ​

常用于展示鼠标 hover 时的提示信息。

基础用法 ​

基础的文字提示用法。

<template>
  <Tooltip content="这是一段提示文字">
    <Button>鼠标移入查看提示</Button>
  </Tooltip>
</template>

<script setup>
import { ref } from "vue";
import Button from "@/components/Button/Button.vue";
import Tooltip from "@/components/Tooltip/Tooltip.vue";
</script>

API ​

属性 ​

参数说明类型可选值默认值
content显示的内容string——
placementTooltip 的出现位置stringtop/top-start/top-end/bottom/bottom-start/bottom-end/left/left-start/left-end/right/right-start/right-endbottom
trigger触发方式stringhover/clickhover
transition过渡动画名称string—fade
openDelay延迟出现，单位毫秒number—0
closeDelay延迟关闭，单位毫秒number—0
